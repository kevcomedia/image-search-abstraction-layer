Image Search Abstraction Layer

Usage

Image search

/api/v1/imagesearch/search-term[?offset=number]

Examples

/api/v1/imagesearch/tuna%20sandwich

/api/v1/imagesearch/funny%20tuna?offset=2

Output

An array of ten results with the following details:

imageUrl
A URL that points to the image.
altText
The image's alt text in the web page it appears in.
pageUrl
A URL that points to a page that contains the image.

Recent searches

/api/v1/recent

Output

An array of the last ten search terms and their respective timestamps

query
The search term.
timestamp
The date and time that search was made.
